# Skill Idea: Create project note

**Goal:** Create a clean project note which uses the existing template and fills makes a clear project objective and scope. 
**Trigger Phrases:** Create a new project for "X"
**Required Tools:** none
**Workflow Steps:**
1. Copy template
2. Fill it out, if there is not enoiught information, prompt the user to provide any remaing info. 
3. Create individual tasks in the kanban board with the correct tag and project note link.
