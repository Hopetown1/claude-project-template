# Skill Idea: TDD

**Goal:** Implement a feature using test driven development from an existing PRD. 
**Trigger Phrases:** Implement "X" PRD
**Required Tools:** none
**Workflow Steps:**
1. Define tests
2. Iterate on code until it passes the tests
3. Human review step
4. Cleanup / refactor
